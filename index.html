👋 Welcome to the Pokémon Binder Spread Builder

A no-account-needed tool for building virtual binder spreads with real Pokémon TCG cards — plus fan art, collection tracking, and print-at-home support.

🔍
Search 17,000+ cards by name, set, era, rarity, artist, energy type, generation, or vibe (Sleepy, Spooky, Cute…).
📖
Build binder spreads up to 4×4 pockets per page. Drag cards into any slot — empty pockets are real drop targets.
🎨
Add fan art from any URL. Paste, crop, resize across 1×, 2-wide, 2-tall, or 4-pocket layouts.
✓
Track your collection & wishlist across multiple binders. The "Not Owned" view doubles as a shopping list.
🖨
Print at real card size (2.5″ × 3.5″). Queue multiple items and dense-pack onto letter paper to save sheets.
💾
Export full backups to JSON — binders, collection, wishlist, and fan art URLs — for long-term storage outside the browser.
Privacy: Everything saves to your browser's local storage. No accounts, no tracking, no servers. Card data comes from pokemontcg.io's public API.

🖨 Print Queue

Items you've added to the queue. They print together on letter-size paper using the same dense-packing layout as Batch Print, so multiple cards/art share one sheet.

📚 Manage Binders

Export downloads everything — all binders, your collection, wishlist, and fan-art URLs — to one JSON file you can save anywhere (Google Drive, iCloud, email, USB). Import restores from that file. Keep regular backups so your work survives browser data clears.

Without a key, the app shares a global anonymous rate limit (~30k requests/day across all users). With a free key from dev.pokemontcg.io, you get 20k/day per key — useful if you share this app with friends. Sign up takes ~30 seconds. Your key is stored only in your browser.

🖨 Batch Print Items

Cards and fan art pack tightly onto letter-size paper so you don't waste sheets. Each pocket prints at exactly 2.5" × 3.5"; multi-cell fan art scales accordingly. Set print scale to 100%, cut along dashed lines.
